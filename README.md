# 🍽️ Meals App

A Flutter mobile application that allows users to browse meals, explore categories, and manage their favorite meals with a smooth and responsive UI.

---

## 📱 Features

- Browse meals by category
- View detailed meal information
- Add meals to favorites ❤️
- Remove meals from favorites
- State management using Riverpod
- Clean UI built with Material 3
- Custom typography using Google Fonts

---

## 🛠️ Tech Stack

- **Flutter** – Cross-platform mobile development
- **Riverpod** – State management
- **Material 3** – Modern UI design
- **Google Fonts** – Custom text styling

## 🎯 Real-World Use Cases (No Backend Required)

Even without a server, this type of app has practical real-life applications:

### 1. Personal Meal Planner

Users can:

- Save their favorite meals
- Quickly decide what to cook
- Organize meals for the week

👉 Useful for individuals trying to stay consistent with diet plans.

---

### 2. Offline Recipe Book

- Store recipes locally
- Access meals without internet
- Ideal for areas with limited connectivity

👉 Perfect for travelers or users in low-network regions.

---

### 3. Gym & Fitness Diet Tracking

- Save preferred high-protein or low-carb meals
- Quickly access diet-friendly options

👉 Helpful for fitness enthusiasts maintaining structured eating habits.

---

### 4. Small Food Business Catalog

- Display available meals to customers
- Show menu categories
- Highlight popular or favorite items

👉 Can be used by small food vendors without needing a full backend system.

---

### 5. Family Meal Organizer

- Families can decide meals together
- Keep track of favorite dishes
- Simplify grocery planning
